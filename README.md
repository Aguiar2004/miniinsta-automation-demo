# 🚀 MiniInsta Automation Demo

Projeto full-stack que simula um fluxo de automação estilo Instagram, integrando backend em Python com frontend em HTML/CSS/JavaScript.

---

## 🎯 Objetivo

Demonstrar:

- Integração entre Excel e aplicação web
- Geração dinâmica de dados (targets.json)
- Automação de navegador com Playwright
- Organização de projeto full-stack
- Execução automatizada com um único comando

---

## 🛠 Tecnologias Utilizadas

- Python
- Playwright
- Pandas
- OpenPyXL
- HTML5
- CSS3
- JavaScript
- HTTP Server
